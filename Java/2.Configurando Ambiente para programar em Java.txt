1° passo - Instalação do JDK, baixar e descompactar no diretório C:/Java (Obs. é preferivel versões javas com a sigla lts = long time suport ou suport de longo prazo);
2° passo - Configurar variáveis de ambiente
*pesquisar variáveis de ambiente na pesquisa do Windows;
*clicar em variáveis de ambiente que fica na aba avançado;
*caso não haja criar variável com o nome exato de JAVA_HOME;
*o diretorio da variável JAVA_HOME sera o local de instalação do seu JDK;
*localizar a variável de ambiente chamada path e edita-la;
*ao editar a variável path clicar em novo pra criar uma nova chave;
*na nova chave acrescentar também o local de instalação do seu JDK porem ira acrescentar no final do diretorio um \bin;
*mover a nova chave criada na variável path para cima até ela ficar em primeiro lugar para ele ter prioridade e evitar conflitos com outras versões do java;
3° passo - verificar a instalação do java
*abrir cmd
*digitar java -version e ira trazer na tela a versão do java instalado;
